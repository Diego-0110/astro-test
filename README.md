# ASTRO TEST

This project is just a test of the main features of Astro Framework.

- Astro files (.astro).
- Astro islands.
- View transitions.
- Endpoint.
- UI frameworks intergrations.

## HOW TO USE

### Dependencies Installation.

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The app will be accessible at http://localhost:4321/
